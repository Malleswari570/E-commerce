# AI Studio App

## Description
This repository contains documentation for my AI Studio application.

## Run Locally

### Prerequisites
- Node.js

### Installation
1. Install dependencies
   ```bash
   npm install
   ```

2. Create a `.env.local` file and add:
   ```
   GEMINI_API_KEY=YOUR_API_KEY
   ```

3. Start the application:
   ```bash
   npm run dev
   ```

## Technologies Used
- JavaScript
- Node.js
- Gemini API
